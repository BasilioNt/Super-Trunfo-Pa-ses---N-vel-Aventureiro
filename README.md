### Jogo Super Trunfo Países ###

O jogador pode escolher qual atributo será usado para comparar as cartas.

Os atributos disponíveis são:
1 - Nome do país   
2 - População
3 - Área
4 - PIB
5 - Número de pontos turísticos
6 - Densidade demográfica

A regra geral é: vence a carta com o maior valor no atributo escolhido. 
Porém, para a Densidade Demográfica, a regra inverte: vence a carta com o menor valor.
